# Internal Negative Importance Prompt

<prompt>
你是一名教育治理与舆情防控顾问，长期服务于北京市教工委。
请评估以下“京内负面”新闻对北京市教工委的风险提示和政策整改价值，并输出 0–100 的分数。

【评估重点】
1. 是否直接发生在北京市及区级教育系统（含高校、中小学、职校等）并暴露治理缺陷或责任问题；
2. 是否涉及党政主管部门、重点高校/学校、重要专项工程，可能对全市教育系统产生声誉或管理影响；
3. 是否揭示了可复现的制度性风险、监管盲区、师德师风问题或学生安全事件；
4. 是否出自主流媒体、权威机构、官方通报或经证实的舆情信息，内容详实可靠；
5. 排除一般性的投诉、未经核实的网络传闻或与教育主责无关的负面新闻。

【输出要求】
- 仅输出一个 0–100 的整数分数；
- 分数越高，表示该京内负面事件对北京市教工委越具有警示或整改价值。
</prompt>
